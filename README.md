# Puzzle Suite

A browser-based worksheet generator that produces vocabulary notes, word search, crossword, and word scramble puzzles from a list of words and clues. Exports to a single vector PDF.

## Quick Start

```bash
# Serve locally (required — ES modules don't load from file://)
python3 -m http.server 8082
# then open http://localhost:8082/puzzle-suite.html
```

## Build

```bash
bash build.sh
# Bundles all JS via esbuild → bundle.js (~80 KB minified)
```

After rebuilding, bump the cache-bust query param in `puzzle-suite.html`:
```html
<script src="bundle.js?v=N"></script>
```

## Features

| Page | Description |
|------|-------------|
| **Notes** | Vocabulary term/definition table. Optional matching mode shuffles definitions. |
| **Word Search** | Configurable grid (10–25). Diagonal and backwards options. |
| **Crossword** | Auto-placed intersecting words. Two-column ACROSS/DOWN clues. Auto-scales to fit one page. |
| **Word Scramble** | Randomised letter order with optional first-letter hint. |
| **Answer Keys** | All four puzzles on one key page, fit to a half-page grid each. |

### Settings
- **Title / Subtitle** — shown on every page header; emoji fully supported in PDFs
- **Title Scale** — shrink or enlarge the header title independently of body text
- **Paper Size** — A4 (210 × 297 mm) or US Letter (215.9 × 279.4 mm)
- **Watermark** — upload a PNG/JPG logo; set opacity
- **Font Family** — Inter, Roboto, Lora, Comic Neue
- **Global Font Scale** — scale all body text up/down
- **Per-page scales** — fine-tune notes, word bank, clue, scramble, and key fonts independently
- **Per-page toggles** — Show Word Bank (crossword), Use Clues (word search), etc.; update preview instantly
- **Bulk export** — generate N unique sets in one PDF

### Quality-of-Life Features
- **Immediate preview updates**: Settings changes apply to preview instantly
- **Color-coded word status**: Sidebar word list shows placement status per puzzle page
- **Undo / Redo**: Full undo history for word list changes (Ctrl+Z / Ctrl+Shift+Z)
- **AI word generation**: BYOK support for Google Gemini, Groq, OpenAI, Anthropic, OpenRouter
- **Responsive design**: Works on desktop and tablet; sidebar collapses on mobile
- **Instruction emoji**: All puzzle pages show emoji-prefixed instructions (📋 📚 🔍 ✏️ 🔀)

## Payment Server (optional)

License purchases and validation require the Express server in `server/`.

```bash
cd server
cp .env.example .env       # fill in Stripe keys, SMTP creds, ADMIN_SECRET
npm install
npm start                  # http://localhost:3001
```

See [`server/README.md`](server/README.md) for full setup, all API endpoints, and deployment options.

The frontend app degrades gracefully — if the server is unreachable it silently stays on Free tier.

## Project Structure

```
puzzle-suite.html   Main app shell & settings sidebar
puzzle-suite.css    All styles (CSS @layer: base › layout › components › pages › utils)
main.js             Entry point; wires all events; exposes window API
bundle.js           Minified output (generated by build.sh)

core/
  state.js          Single-source state + DOM sync (syncSettingsFromDOM / applyStateToDOM)
  storage.js        localStorage persistence (debounced saveState)
  history.js        Undo/redo stack (50-entry max, word list only)

renderers/
  notes.js          HTML preview for notes/vocabulary page (matching mode aware)
  wordSearch.js     HTML preview for word search
  crossword.js      HTML preview for crossword (two-column, auto-scale)
  scramble.js       HTML preview for word scramble
  keys.js           HTML preview for answer key page
  wordList.js       Sidebar word list with page-aware status coloring

workers/
  workerBridge.js         Web Worker interface (inlined as Blob URL)
  generation.worker.js    WS, CW, scramble generators

pdf/
  pdfExport.js            Orchestrates PDF export (loops sets, calls drawHeader + drawXxx)
  pdfHelpers.js           buildCtx(), drawHeader(), emoji canvas fallback
  pdfDrawNotes.js
  pdfDrawWordSearch.js
  pdfDrawCrossword.js
  pdfDrawScramble.js
  pdfFonts.js             Lazy-loads Inter/Roboto/Lora/Comic Neue from CDN

ui/
  toast.js          showToast() utility
  modal.js          Bulk import modal helpers
  darkMode.js       Dark mode toggle
  sidebar.js        Sidebar tabs and resize handle
  zoom.js           Preview zoom controls
  pageOrder.js      Drag-to-reorder export page list
  dropZone.js       File drag-and-drop

ai/
  aiGenerate.js     BYOK AI word generation (Gemini, Groq, OpenAI, Anthropic, OpenRouter)

license/
  licenseManager.js Frontend license manager (tier state, validation cache, Stripe redirect)

import-export/
  importWords.js    Parse text/JSON imports
  exportConfig.js   Download state as puzzle-config.json

server/             Payment server (Node.js + Express — see server/README.md)
  index.js          Express app entry point
  db.js             SQLite database (licenses + event audit log)
  email.js          Nodemailer license delivery emails
  routes/
    checkout.js     POST /api/checkout/session, GET /api/checkout/plans
    webhook.js      POST /api/webhook (Stripe events)
    license.js      GET /api/license/validate
    admin.js        Admin CRUD API (/api/admin/*)
  admin.html        Self-contained admin dashboard
  .env.example      Configuration template
```
